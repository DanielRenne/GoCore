Simple Alerts

Provide contextual feedback messages for typical user actions with the handful of available and flexible alert messages. We provide two alerts style for you to choice by switching class .toast-flat and .toast. Wrap any text and an optional dismiss button in .toast and one of the four contextual classes (e.g., .toast-info) for basic alert messages.
